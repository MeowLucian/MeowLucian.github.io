Static 關鍵字 (Static Keyword)

Static 關鍵字 教學與筆記。

三大功能

隱藏(私有化)

存取的範圍：只在定義的文件檔內，不具跨文件檔共享。也可以避免與其他人寫的函式名稱衝突的問題。

  • 通常寫在 .cpp 檔：
  • 若宣告 static 在 header 檔：則所有 include header 的 .cpp 檔 都會產生 static。

持久生命週期

一但變數生成，它就會一直存在記憶體之中；直到程式結束，才會從記憶體消逝。
